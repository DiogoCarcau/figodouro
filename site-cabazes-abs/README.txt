LOJA (PROJETO ACADÉMICO) — ATLÂNTICO BUSINESS SCHOOL

Funcionalidades:
- Catálogo de produtos (amêndoa, figo, noz, azeitona, leguminosas, azeite 2L/5L) + cabaz.
- Pesquisa no topo (campo "O que procura?").
- Carrinho (adicionar, aumentar/diminuir, remover).
- Finalizar compra (simulado): gera nº de encomenda e mostra página de sucesso.
- Preços:
  - Produtos por peso: €/Kg
  - Azeite: €/L
  - Cabaz: €/UN

Testar localmente:
- Abrir a pasta no VS Code.
- Recomendado: extensão "Live Server" -> abrir index.html com "Open with Live Server".
- Alternativa: abrir index.html no browser.

Publicar (GitHub Pages):
- Criar repositório e enviar estes ficheiros para a branch main.
- GitHub -> Settings -> Pages -> Deploy from a branch -> main /(root).
